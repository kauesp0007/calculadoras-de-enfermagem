Medidas de jeringas para la aplicación de medicamentos por vía subcutánea - Jeringa de Insulina de 1 mL:

Ejemplos:

100 UI equivale a 1 mL o 1 cc

50 UI equivale a 0,5 mL o 0,5 cc

30 UI equivale a 0,3 mL o 0,3 cc

(Dé preferencia a las jeringas de Insulina con capacidad para 100 UI)

Atención: 1 UI de Insulina equivale a 0,01 mL > 10 UI equivale a 0,1 mL

ATENCIÓN: LA INSULINA ES UN MEDICAMENTO DE ALTA VIGILANCIA (MAV), REALICE LA DOBLE VERIFICACIÓN (DOUBLE CHECK) JUNTO A SU ENFERMERO.
