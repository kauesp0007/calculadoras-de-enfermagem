Политика доступности

Наша приверженность доступности

Мы стремимся обеспечить равный доступ к информации и услугам для всех пользователей, включая людей с различными ограниченными возможностями. Наш сайт разработан в соответствии с международными стандартами веб-доступности.

Стандарты соответствия

Мы стремимся соответствовать следующим стандартам и рекомендациям:

  • Руководство по доступности веб-контента (WCAG) 2.1, уровень AA
  • Европейский стандарт EN 301 549
  • Закон США о восстановлении и реабилитации инвалидов (Section 508)
  • Бразильский закон о включении инвалидов (Lei Brasileira de Inclusão)

Функции доступности

Навигация с клавиатуры

Все функции сайта доступны через клавиатуру с использованием стандартных сочетаний клавиш.

Совместимость с экранными считывателями

Сайт совместим с популярными экранными считывателями, включая JAWS, NVDA и VoiceOver.

Альтернативный текст

Все изображения имеют описательные альтернативные тексты для пользователей с нарушениями зрения.

Контрастность цветов

Мы обеспечиваем достаточный контраст цветов для пользователей с нарушениями зрения.

Адаптивный дизайн

Сайт автоматически адаптируется к различным размерам экрана и разрешениям.

Управление фокусом

Четкое визуальное выделение активного элемента при навигации с клавиатуры.

Технические характеристики

Поддерживаемые технологии

  • • HTML5 семантическая структура
  • • CSS3 с использованием Flexbox и Grid
  • • JavaScript с прогрессивным улучшением
  • • ARIA атрибуты для улучшения доступности
  • • Responsive design для всех устройств

Специальные возможности

Для пользователей с нарушениями зрения

Мы предоставляем поддержку экранных считывателей, возможность увеличения шрифта и высококонтрастные цветовые схемы.

Для пользователей с нарушениями слуха

Вся информация представлена в текстовом формате, без зависимости от аудиоконтента.

Для пользователей с ограниченной подвижностью

Полная навигация с клавиатуры, поддержка различных устройств ввода и настраиваемые интервалы времени.

Для пользователей с когнитивными нарушениями

Четкая структура контента, простой язык и последовательная навигация.

Тестирование доступности

Мы регулярно проводим тестирование доступности с использованием:

  • Автоматизированные инструменты тестирования (axe-core, WAVE)
  • Ручное тестирование с экранными считывателями
  • Тестирование навигации с клавиатуры
  • Проверка цветового контраста
  • Обратная связь от пользователей с ограниченными возможностями

Постоянное улучшение

Мы стремимся к постоянному улучшению доступности нашего сайта. Это включает регулярный мониторинг новых стандартов, технологий и лучших практик. Мы также активно ищем обратную связь от наших пользователей для выявления областей, требующих улучшения.
