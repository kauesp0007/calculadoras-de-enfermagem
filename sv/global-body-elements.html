Tillgänglighetsgenvägar (Åtkomstnyckel)

Använd följande tangentkombinationer för att snabbt komma åt tillgänglighetsfunktioner:

  • Alt + Skift + I:Gå till startsidan
  • Alt + Skift + C:Gå till huvudinnehåll
  • Alt + Skift + A:Gå till tillgänglighetsverktygsfältet
  • Alt + Skift + T:Tillbaka till toppen av sidan
