Pintasan Keyboard dan Navigasi

Pintasan Aksesibilitas (Accesskey)

Gunakan kombinasi tombol berikut untuk mengakses fitur aksesibilitas dengan cepat:

  • Alt + Shift + I: Pergi ke Beranda (Home)
  • Alt + Shift + C: Pergi ke Konten Utama
  • Alt + Shift + A: Pergi ke Bilah Aksesibilitas
  • Alt + Shift + T: Kembali ke Atas Halaman
