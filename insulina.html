Cálculo de Insulina

Calculadora de Insulina

Medidas de seringas para aplicação de medicamentos por via subcutânea - Seringa de Insulina de 1 mL:

Exemplos:

  • 100 UI equivale a 1 mL
  • 50 UI equivale a 0,5 mL
  • 30 UI equivale a 0,3 mL
  • 10 UI equivale a 0,1 mL

(Dê preferência a seringas de Insulina com capacidade para 100 UI)

ATENÇÃO: INSULINA É UMA MEDICAÇÃO DE ALTA VIGILÂNCIA (MAV), REALIZE A DUPLA CHECAGEM (DOUBLE CHECK) JUNTO AO SEU ENFERMEIRO.
